A space calculator web-app.

## Building from source

Requirements:

- Go@^1.23
- Deno@^2.0

> [!TIP]
> First setup:
>
> 1. Run `deno task init:build`.
> 2. Change the `.env` file.
> 3. Run `deno task serve`.
